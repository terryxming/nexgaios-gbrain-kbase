# AIWork Knowledge Base

本知识库用于沉淀 Terry / NexGaios 的 AI 工作体系，包括 AI 工具、agent 设计、自动化工作流、官方资料、文献、行业新闻和实验记录。

## 收录范围

- AI 工具、平台、模型和技术方案资料
- Agent、workflow、prompt、MCP、RAG 等能力模块
- 官方文档、教程、最佳实践和配置经验
- 行业新闻、产品发布、趋势观察和竞品研究
- AI 项目实验、评测、复盘和方法论沉淀

## 不收录

- API key、账号密码、内部凭据等敏感信息
- 未验证的零散观点，除非明确标记为草稿或待验证
- 与 AI 工作无关的业务资料

## 当前目录说明

- `00 - inbox/`：临时收集，待整理。
- `官方资料/`：官方文档、产品说明、API 文档摘要和配置记录。
- `能力模块/`：按能力沉淀，如 GBrain、MCP、RAG、Agent、Prompt Engineering 等。
- `文献/`：论文、研究报告、技术文章和阅读笔记。
- `行业新闻/`：行业动态、产品发布、融资、趋势和重要观点。

当前目录是暂行结构，可以随着学习和项目推进调整。项目或能力模块相关材料优先放在对应主题目录下集中维护。

## 维护规则

- 官方资料尽量保留来源链接、访问日期和适用版本。
- 文献和新闻应注明发布时间、来源、核心结论和对业务的启发。
- 实验记录要写清楚目标、方法、输入、结果、结论和下一步。
- 能力模块下可以按项目继续拆分子目录。
- 临时内容先放 `00 - inbox/`，定期整理到对应目录。

## GBrain 使用说明

GBrain 会同步本知识库，用于回答 AI 工具选型、工作流设计、agent 架构、学习路径和项目复盘相关问题。写作时请尽量保留工具名称、模型名称、版本、来源链接、日期和明确结论，方便 AI 检索、引用和综合回答。
