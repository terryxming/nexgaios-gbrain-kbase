# 2026-06-22 GBrain 产品决策时间线与流水线复盘

## 0. 文档定位与读法

本文件不是日报，也不是一次性总结。

本文件是 GBrain 产品决策与事故复盘的 raw 记录。它采用时间线优先结构：

```text
T = 时间节点
O = 该时间节点产生或更新的未闭环项
```

示例：

```text
T-004-O1 = T-004 时间节点产生的第 1 个未闭环项
```

因此：

1. `决策与复盘时间线` 是主表。
2. `当前未闭环清单` 是从时间线派生出的索引，不是另一套独立编号。
3. `当前有效判断`、`当前方案`、`规则影响面`、`Agent 使用约束` 都必须绑定来源时间节点。
4. 没有来源时间节点的判断，不应被 Agent 当成有效规则。

当前文件位于 `00 - raw/00 - inbox`，只表示已被记录，不表示已经完成路由、编译、验收或成为 active schema。
