# Amazon Knowledge Base

本知识库用于沉淀 Terry / NexGaios 的 Amazon 业务知识，包括 Listing、广告、市场、政策、运营经验、平台规则和项目资料。

## 收录范围

- Listing 优化方法、案例、素材和关键词研究
- 广告投放策略、复盘、数据观察和优化方法
- 市场调研、竞品分析、品类趋势和用户洞察
- Amazon 政策、合规要求、风险提醒和案例
- Amazon 业务项目相关资料、会议、决策、数据和复盘

## 不收录

- 账号密码、API key、店铺后台凭据等敏感信息
- 未脱敏的个人隐私或客户敏感数据
- 与 Amazon 业务无关的临时资料

## 当前目录说明

- `00 - inbox/`：临时收集，待整理。
- `Listing/`：标题、五点、描述、A+、关键词等 Listing 相关知识。
- `广告/`：广告结构、投放策略、ACOS/TACOS 优化和复盘。
- `市场/`：品类、竞品、用户、趋势研究。
- `政策/`：平台规则、合规、风控和案例。

当前目录是暂行结构，可以随着业务组织方式调整。项目相关材料优先跟随项目目录集中维护。

## 维护规则

- 一个主题尽量独立成一篇 Markdown。
- 重要项目可以在对应业务目录下单独建文件夹。
- 项目内的会议、决策、数据、素材、产出和复盘尽量放在同一项目目录下。
- 临时内容先放 `00 - inbox/`，定期整理到对应目录。
- 重要事实尽量写明日期、来源、适用范围和结论。

## GBrain 使用说明

GBrain 会同步本知识库，用于回答 Amazon 业务相关问题。写作时请尽量使用清晰标题、日期、项目名、产品名、店铺/市场/平台规则等关键实体，方便 AI 检索、引用和综合回答。
